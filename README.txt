Deployment checklist

1. Confirm `site-config.js`, HTML canonical tags, Open Graph tags, `robots.txt`, and `sitemap.xml` all point at the live production URLs.
2. Deploy the full folder contents, including `assets`, to the live site root.
3. Verify the site property in Google Search Console.
4. Submit `sitemap.xml` in Search Console.
5. Test `robots.txt` is reachable and correct.
6. Confirm there are no broken internal links.
7. Confirm the public App Store CTA points at the live product page and renders the badge correctly.
8. If the App Store URL changes later, recheck that the CTA state matches the configured links.
